Describe use case for parenthese () in JavaScript

In JavaScript parenthese are used when defining a JavaScript function expression, when calling or running a function, passing values to a function to be processed or used and declaring a function that can run automatically also known as an Immediately Invokable Function Expression or IIFE.

For example, when defining a new function, parenthese are placed after the function keyword to declare an annonymous function function(), or placed after a function name function doSomeThing(). When defining a self invoking function or IIFE, parenthese are used to wrap the function declaration, after the function keyword or the function name and at the end of the declaration (function(){ statement })().

When calling or running a function parenthese are used following the function keyword or function name doSomeThing() and when passing values into a function doSomeThing(value).

Describe the use case for brackets [] in JavaScript

Brackets are used to define a new array or list which can be used to store information together in a comma separate list let letters = [A,B,C,D,E,F,G]. Brackets are also used to access all the items stored in the list using the items position within the list, for example to get the letter 'A' we use letters[0].

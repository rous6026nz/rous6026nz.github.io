What is HTML, CSS and the DOM(Document Object Model)?

HTML stands for Hyper Text Markup Language and is the standard language for building webpages for website, and describes and outlines the structure. HTML provides the basic building blocks for Web Designers and Developers to build websites and web applications for the web.

CSS stands for Cascading Stylesheets and is used to provide visual styling to the HTML docuemnt and elements within the document, and is used to describe the presentation of an HTML document. CSS applies styles to elements in a cascading fashion meaning the visual styles are applied based on the most recent style definition.

The DOM or Document Object Model is an object-orientated representation of a webpage. Before a webpage is displayed to the user, the browser parses the HTML document and turns it into the DOM which then can be with a scripting language such as JavaScript.

What is meant by boxifying design?

Boxifying is the process of breaking a webpage design down into rectangular sections or boxes. Boxifying a design helps with building out the basic structure and layout of a webpage and can reduce the amount of elements in the HTML markup.

What is the box model?

The Box Model is a box representation of an HTML element, the Box Model displays infomation about an HTML element including the margin, padding, border, width and height style values in a visual diagram.

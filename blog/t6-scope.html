What 'scope' is, and how it works in JavaScript?

In the JavaScript programming language, "Scope" refers to the availability of a variable or function within a program. Variables and functions can be defined in two scopes within a JavaScript program, local and global. Depending on the context of it's creation determines accessibility of the variable or function within the code.

Global Scope

When a variable or function is defined in the global scope of a JavaScript program, it is accessible anywhere in the program.

Local Scope

A variable or function defined within a function has a local scope, it is only accessible within the context of which it was created in. In JavaScript, if a variable is created within the local scope of a function without the uses of the "var" keyword, the program will look up a level out of the local scope of the function it is being created in, to see if the variable has been created up a level.

This will happen until the program has found the first instance of the variable creation or has reached the global scope of the program. If no instances of the variable are found, it will create the variable within the global scope of the program.

Lexical Scope

Also refered to as static scope, is when you have nested functions (functions created within another function), the inner functions are able to access variables within it's containing function.

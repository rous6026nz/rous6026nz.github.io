What is grid based design?

Grid based design is using vertical and horizontal guidelines to place elements in a design and help with aligning elements.

Whats all the hype about responsive webpages?

Responsive webpages have acquired much attention over the past few years, as it provides a way for web designers and developers to design and build websites that are mobile-friendly, improve the way it looks on devices with both large and small screens, and can also help improve your sites search engine ranking.

What is semantic structure?

Semantic structure is the use of HTML markup to help outline the webpage and best describe the type or meaning of information on the webpage.

Semantic markup is processed by the browser and is also used by spiders to crawl for information.
